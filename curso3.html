Atividade de Fixação

Em um banco de dados relacional, cada registro em uma tabela deve ter um valor exclusivo em sua chave primária.



O modelo NoSQL é uma abordagem que permite o armazenamento e recuperação de dados não estruturados.



Índices em um banco de dados são utilizados para melhorar o desempenho das consultas, mas podem diminuir a velocidade das operações de atualização.



Normalização em banco de dados é o processo de organizar os dados para minimizar a redundância e dependência.



Um banco de dados NoSQL do tipo chave-valor é adequado para casos em que a estrutura dos dados é altamente relacional e complexa.
