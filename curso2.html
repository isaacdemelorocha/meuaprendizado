Atividade de Fixação

HTML5 introduziu uma nova tag chamada 'article' para representar um conteúdo independente que pode ser distribuído e reutilizado.



CSS3 introduziu a propriedade flexbox para facilitar o layout de elementos em uma única dimensão, seja em linha ou coluna.



HTML5 inclui uma API chamada localStorage que permite armazenar dados no navegador de forma persistente.



Em CSS3, a propriedade box-shadow permite adicionar sombras a caixas HTML, oferecendo controle sobre a cor, desfoque e posição da sombra.



HTML5 introduziu a tag 'font' para fornecer uma maneira mais semântica de estilizar o texto em comparação com as versões anteriores do HTML.
