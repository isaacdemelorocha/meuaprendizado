Avaliação para Certificação

A análise exploratória de dados é uma etapa importante na ciência de dados, que envolve a visualização e a compreensão dos padrões nos dados.



Na ciência de dados, a etapa de pré-processamento de dados inclui tarefas como limpeza de dados, preenchimento de valores ausentes e normalização.



A correlação entre duas variáveis indica a presença de uma relação de causa e efeito entre elas.



A mineração de dados é uma técnica que visa descobrir padrões e conhecimentos úteis a partir de grandes conjuntos de dados.



A aprendizagem de máquina é um subcampo da ciência de dados que se concentra no desenvolvimento de algoritmos que permitem computadores aprenderem.
